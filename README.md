# Netanel Asraf - Personal Portfolio
## About The Project

This is a highly optimized, fully responsive web application built with modern web technologies. I focused on clean aesthetics, a seamless user experience, and robust code architecture. 

**Key Features:**
- **Custom Interactive UI**: Features a dynamic mouse-tracking spotlight background effect that gives the page an immersive feel.
- **Modern Tech Stack**: Built entirely with Next.js 15 (App Router) and React 19 for ultra-fast performance and SEO.
- **Vanilla CSS Engineering**: Avoiding heavy UI libraries, I built the entire design system from scratch using CSS Modules. This includes custom CSS variables, responsive grid/flex layouts, and smooth micro-animations.
- **Type Safety**: Strictly typed with TypeScript.
- **Continuous Deployment**: Fully automated CI/CD pipeline using GitHub Actions to seamlessly build and deploy static exports directly to GitHub Pages.

## Built With
* [Next.js](https://nextjs.org/)
* [React](https://reactjs.org/)
* [TypeScript](https://www.typescriptlang.org/)
* Vanilla CSS Modules
* GitHub Actions

## Getting Started Locally

If you'd like to check the code or run it locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/Netanel-Asraf/Netanel-Asraf.github.io.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
